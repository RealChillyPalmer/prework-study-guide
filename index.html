HTML

  • HTML contains the foundational elements of a web page
  • HTML elements are seperated by a '<..>' beginning and ' ending tag
  • The Head Element contains information for the browser about the webpage
  • The body element represents the visible content shown to the user
  • The style elemet defines the CSS styles associated with a website
  • The base element defines the URL for a webpage
  • The link element connects an external resource to the HTML doc
  • The meta element defines metadata such as the character set, description, keywords, author, and viewport.
  • h1, h2, h3, h4, h5, and h6 elements represent levels of heading. *There should only be 1 h1
  • The p element represents a paragraph or block of text.
  • The ul, ol, and li elements represent unordered lists, ordered lists, and list items
  • The ul elements contain unordered or bulleted lists where each list item is tagged with the li element
  • The img element contains information about images that are displaued on the webpage and can contain various elemnts
  • The src attribute is the most important attribute as it defines the location of the image in an img element
  • The br element creates a line of empty space/line break between two blocks of content. *written as <'br /'>
  • The a element "Anchor" creates links to a webpage. The href attribute points to the URL for the link

CSS

  • CSS is the language used to style the HTML for viewing in browser
  • There are 3 CSS styles to consider when styling HTML: Inline, Internal, and External
  • Inline CSS styles HTML directly in the HTML file.
  • Inline styles apply to a specific HTML tag, using a style attribute with a CSS rule to style a particular page element only
  • Inline Styles are the least flexible of the 3 styles but are useful for quick permanent changes
  • Internal CSS embeds a entire stylesheet directly in the HTML file using a "style" element in the "head" elemet
  • Internal CSS rules only apply to page with rules in head element
  • External CSS is most common way to style HTML.
  • External CSS allows developeers to keep all CSS rules in a seperate file, making design easier
  • External CSS uses a "link" element placed in HTML "head" element
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Margin is outside the border while padding is inside of border
  • Box-shadow property adds shadow effect around elemnt frame. Set multiple effects seperated by commas. desribed by X and Y offsets in px "pixels", then blur, then color
  • Color can be adde in CSS as semantic "by name", hex #, etc
  • To apply rule to all elements visible on page the wild card "*" symbol is used.
  • To combine multiple selectors into a single CSS rule : Seperate by comma and next line
  • Use class attributes to create groupings ato apply CSS styling to certain elements but not all.
  • The command ."class name" : Designates class selectior.

Git

  • The git command must precede every Git statenebt
  • ls : lists files in current folder
  • cd : changes directories. cd .. returns to previous directory
  • pwd ("print working directory") : will print current directory to the terminal
  • git status : checks current branch
  • git checkout Main : returns to main branch
  • git checkout -b (branch name) : creates a new branch and switches to it
  • git add -A : adds all modifications to branch staging area. Where add is the command and -A flag indicates all changes
  • git commit -m "notes" : commits staged files to current branch
  • git pull origin main : pulls main branch from remote to local to ensure they are in sync
  • git push origin "current branch name" : Push local changes to remote Github branch
  • eval "$(ssh-agent -s) : starts the ssh agent. sshe agent stores key for use in git
  • code . : Opens VS code and any file in current directory. The code command just opens VS Code

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • To declare a variable in JavaScript we may use the "const" keyword first or let keyword followed by a unique variable and assignemnt operator
  • let allows dev to reassign the value of a variable over time.
  • undefined indicates the absence of a value, while null indicates the absence of an object
  • Null is a keyword but Undefined is a normal indentifier and global property
  • Boolean type represents a logical entity and can be either "true" or "false"
  • NaN = "Not a Number" and is only JS value that is not equal to itself
  • BigInt type is a numeric primitice that can represent integers with arbitrary magnitude.
  • only forbidden operator (+, -, *...) is >>>
  • String type represents textual data and is encoded as a sequence of integer values.
  • JavaCript strings are immutable (which means it cannot be modified once created
  • A symbol is a unique and immutable primitive value. Symbols are called "atoms"
  • "console.log("name of argument")" will output the value to the web console
  • When the computer encounters an if statement it will check to see if (condition) is truthy, then carry out {action requested}. If not truthy action is ignored and pc moves on
  • "else if" adds more conditions to if conditions
  • Strict equality operator (===) checks if 2 values are equal and returns boolean result
  • Strings must be contained in "quotation marks"
  • An array contains many items in one location
  • To work with individual objects in an are the location is called using an index starting with 0. (0,1,2,3...)
  • for loop : uses predictable pattern of indices to perform a task on all items in an arry with a single block of code
  • for loop must contain following 3 things: starting point, condition, iteration amount (amount to change by each time)
  • for loops must be broken by another statement such as conditional "if" for control flow to be interrupted
  • for loops require the array length to dignify when complete
  • 2 req's for JavaScript funtion: Define the function, Call the function
  • A function's name is followed by parentheses which parameters can be placed inside
  • "Math.floor() and Math.random() are both properties of JS math object and can be used together to create a random number
  • Commenting out code : put "/*" before and "*/" after the block of code
